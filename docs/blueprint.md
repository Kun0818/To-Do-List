# **App Name**: GitTodo

## Core Features:

- Add Task: Allows users to input and add tasks to their todo list.
- Mark Complete: Enables users to mark tasks as complete, visually striking them through and graying them out.
- Delete Task: Allows users to remove tasks from the list with a confirmation prompt.
- Data Persistence: Saves task data in the browser's localStorage to persist it across sessions.

## Style Guidelines:

- Primary color: Soft teal (#64CCC5) for a calm and productive feel.
- Background color: Light grey (#F0F0F0), nearly white, providing a clean backdrop.
- Accent color: Coral (#FF7F50), offering a cheerful and contrasting highlight for interactive elements.
- Body and headline font: 'PT Sans' for a modern, readable interface.
- Minimalist icons for task actions (add, delete, complete), using simple line art.
- Clean and straightforward layout with a focus on readability, using a single-column design for tasks.
- Subtle animations on task completion or deletion to provide visual feedback without being distracting.
